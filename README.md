# GoPick – SIT725 Group Project

GoPick is a group project developed for **SIT725 – Application Software Engineering**.  
The project focuses on building a web-based shopping application using a Node.js backend and a MongoDB database, following iterative development across multiple sprints.

---

## Project Overview

The GoPick application provides core e-commerce functionality, including product browsing, cart management, FAQ support, and user account-related features.  
The system is designed with a modular backend structure and a clear separation between routes, controllers, and data models.

Development follows an agile workflow, with sprint-based planning, task tracking via Trello, and version control managed through GitHub.

---

## Key Features

- Product listing and basic product interaction
- Add-to-Cart functionality
- FAQ page with searchable content with feedback feature
- User authentication (planned / partially implemented depending on sprint)
- Incremental feature development across sprints

---

## Technology Stack

- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Frontend:** Server-rendered pages (EJS / HTML, CSS, JavaScript)
- **Version Control:** Git & GitHub
- **Project Management:** Trello
- **Testing (current & planned):**
  - Manual testing during development
  - Postman API testing
  - Basic Mocha unit tests
